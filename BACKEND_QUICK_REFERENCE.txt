                      TROUBLESHOOTING
================================================================================

Port Already in Use:
  $ PORT=3002 npm start
  $ lsof -i :3001
  $ kill -9 <PID>

Database Connection Error:
  $ rm -f dev.db
  $ npm run db:push
  $ npm run db:seed

CORS Issues:
  1. Check FRONTEND_URL environment variable
  2. Verify origin is in getCorsOptions() array
  3. Review server/index.js lines 37-67

Server Not Starting:
  $ npm install
  $ npm run db:push
  $ npm start

Cannot Connect from Network:
  1. Verify PORT is correct (3001)
  2. Check firewall allows port 3001
  3. Use machine IP: http://<your-ip>:3001
  4. Verify server bound to 0.0.0.0 (should show in logs)
